# 异常处理

## 1.异常处理配置
```ini

;开启此项，Yaf会在发生错误的地方抛出异常。
application.dispatcher.throwException=1

;开启此项，如果有未捕获的异常，Yaf将会把它定向到Error controller, Error Action。
application.dispatcher.catchException=1

```
