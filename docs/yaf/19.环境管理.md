# 环境管理

## 1.环境

|环境名称|地址|
|:-----|:---|
|开发环境|127.0.0.1|
|测试环境|192.168.1.58|
|UAT环境|192.168.1.100|
|生产环境|92.168.1.100|

* 新需求->新的开发git分支->多个人在这个分支下开发->开发自测完毕->从master分支打出release上线分支->开发分支代码合并到release分支->测试人员在测试环境测试release分支->在uat环境验收->上线

## 2.差异
* 数据库配置
* 短信验证码
* 第三方服务，如：第三方登录
* 加密公钥私钥

## 3.运行时配置

```ini
yaf.environ=develop
;yaf.environ=test
;yaf.environ=uat
;yaf.environ=product
```
